## 문제

- 백준 온라인저지 1890번

## 풀이

DP를 이용해 풀었으며 d[i][j]는 (i,j)로 가는 경로의 수이며 (i,j)에 있을 때 다음까지 가능한 거리를 계산함을 통해서 진행합니다.

## 코드

```cpp

#include <iostream>

int a[100][100];
long long d[100][100];

using namespace std;

int main(){


	int n;

	cin >> n;

	for(int i=0; i<n; i++){
		for(int j=0; j<n; j++){
			cin >> a[i][j];
		}
	}

	d[0][0] = 1;
	for(int i=0; i<n; i++){
		for(int j=0; j<n; j++){

			if(a[i][j] == 0) continue;

			if(j+a[i][j] < n){

				d[i][j+a[i][j]] += d[i][j];

			}

			if(i+a[i][j] < n){

				d[i+a[i][j]][j] += d[i][j];

			}

		}
	}


	cout << d[n-1][n-1] << '\n';

	return 0;
}

```
